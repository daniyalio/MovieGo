In order to run this project, one needs an api key from www.themoviedb.org.
1- Sign up on www.themoviedb.org
2- Click on the profile, then setting.
3- Go to the api and request for the generation of an api key
4- Now,Go to api.js (located in "services" folder)
5- Copy your api key and paste here
    Sample:
    const API_KEY = "...................";
6- Then all set.
